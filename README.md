# Gesture Particle Matrix

An interactive particle visualization system controlled through hand gestures.

## Features

- Real-time hand tracking
- Gesture-based shape switching
- Dynamic particle effects
- Camera zoom controlled by palm distance
- Built using Three.js and MediaPipe

## Technologies

- HTML
- JavaScript
- Three.js
- MediaPipe Hands

## Project Overview
